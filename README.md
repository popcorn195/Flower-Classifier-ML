## 🚀 Getting Started

### ✅ Prerequisites

Get the training data set on : https://drive.google.com/drive/folders/1rzEFpraXXLLIGPo6F6jNEpeMOdWMgOzN

Install the following:

```bash
pip install tensorflow numpy streamlit
```

Run the app.py using:

```bash
streamlit run app.py
```
